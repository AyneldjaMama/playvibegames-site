The Studio
VIBE
GAMES

One person. Clean mechanics. No BS.

Vibe Games is an indie mobile game studio focused on a single principle: strip the game down to one mechanic and make that mechanic feel exceptional. No gacha, no paywalls, no endless content drip. Just tight, replayable arcade loops built for the five minutes you have between things.

01
One mechanic per game
Every game in the catalog is built around a single input. The depth comes from mastery, not complexity.
02
Free. Actually free.
One optional rewarded ad for an extra life. That's the entire monetization model across every title.
03
Daily challenge in every game
Same seed, global competition. One shot per day. Keeps the catalog alive long-term.
